# Excel 2D RPG Core

Excel VBA で構築された 2D RPGエンジンのコア部分です。  
マップ描画・移動・NPC制御・階段遷移までを含む技術公開版です。

本リポジトリは **技術参考用（非商用）** として公開しています。

## ■ 操作方法

- ↑↓←→：移動
- Xキー：決定
- Zキー：キャンセル

※ 日本語入力モードは OFF で実行してください。

---

## ■ ワールドマップ仕様

- ワールドマップには必ず 1つ "R" を配置してください。
- "R" のセルの右下が座標 (1,1) になります。

---

## ■ プレイヤー人数変更

以下の値を変更してください。

myPlayData.PLAYER_CNT

---

## ■ Known Issues（既知の問題）

- 隊列3人目の挙動に軽微な不具合あり
- ワークシート上に 1～20 の数値が残っていると  
  NPCデータとして誤認識される場合があります
- NPCトラッキングエリアに描画データや軌跡データが入ると  
  NPCが正常に動作しない場合があります

ワークシート編集時は十分ご注意ください。
